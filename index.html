Introducing the Analytics Catalog

The Analytics Catalog aims to streamline the sharing of data knowledge across all end users for analytics by creating a single place for users to find information on analytics products created by the Analytics Foundations pod. This platform will surface definitions on metrics we have built, documentation on where to find and when to use trusted reports that house those metrics, and meta-data & documentation on the models that power those metrics. This catalog is a living platform, and as we continue to build, audit, and release trusted data, we will continue to surface the end to end information here.

  • See our Release Notes for the Latest Updates

Catalog Overview

  • Reporting Portal

    Provides information on where to find critical Mode & Tableau reports and when to use them.

  • Metrics Dictionary

    Provides in depth definitions on metrics and which reports to navigate to, to reference those metrics.

  • Table Metadata

    Provides table descriptions and field definitions to help end users understand when to use core tables in ad hoc queries.
